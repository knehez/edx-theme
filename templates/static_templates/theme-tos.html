MeMOOC Honor Code

By enrolling in a course on MeMOOC you agree to the Honor Code below and any additional terms specific to a class.



By enrolling in an MeMOOC course, I agree that I will:

  • Accept the Rules and Regulations of University of miskolc, which can be found at this link http://www.uni-miskolc.hu/szabalyzatok
  • Complete all mid-terms and final exams with my own work and only my own work. I will not submit the work of any other person.
  • Maintain only one user account and not let anyone else use my username and/or password.
  • Not engage in any activity that would dishonestly improve my results, or improve or hurt the results of others.
  • Not post answers to problems that are being used to assess student performance.


CHANGING THE HONOR CODE

Please note that we review and may make changes to this Honor Code from time to time. Any changes to this Honor Code will be effective immediately upon posting on this page, with an updated effective date. By accessing the Site after any changes have been made, you signify your agreement on a prospective basis to the modified Honor Code and any changes contained therein. Be sure to return to this page periodically to ensure familiarity with the most current version of this Honor Code.

MeMOOC Becsületkódex

Bármely MeMOOC kurzus felvételével a Becsületkódexet elfogadom és magamra nézve kötelezőnek tartom:

  • Elfogadom a Miskolci Egyetem szabályzatait, amelyeket itt olvashatok: http://www.uni-miskolc.hu/szabalyzatok
  • Minden vizsga, beadandó feladat az én munkám és csakis az én munkám. Más munkáját nem adom be sajátomként.
  • Csak egy felhasználói fiókot használok és a felhasználói nevemet, jelszavamat nem adom át másnak használatra.
  • Tevékenységemmel nem bántok meg másokat.
  • Nem válaszolok meg olyan kérdéseket, amelyeket mások értékelésére használnak.


MeMOOC Becsületkódex megváltoztatása

A MeMOOC fentartja a jogot, hogy a Becsületkódexet időről időre megváltoztassa. Minden változtatás a kihirdetéstől fogva hatályos.
Amennyiben a kihirdetés után belépsz a MeMOOV oldalaira, elfogadod a módosított becsületkódexet. Ezért kérjük, hogy rendszeresen olvasd el a Becsületkódex legfrissebb változatát
